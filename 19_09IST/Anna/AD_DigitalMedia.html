Text Encoding

Our devices can only store 'bits' (binary digits), so all text, images, and videos have to be encoded.

Bits only have two values: 0 and 1.

Text encoding is the translation of letters into 8 bits, so that our devices can recieve and process text.

This form of encoding is called ASCII (American Standard Code for Information Interchange).

Bitmaps

A bitmap is a series of pixels that make up a bigger image.

Each pixel is a tiny square that is assigned a colour, and arranged to form a image. A bitmap is made of two values: 0 and 1.

A simple one bit image will only have two colours: white and black (1 and 0).

Different colours can be assigned by increasing the amount of bits, which increases the amount of combinations possible.

Runlength Encoding

Runlength encoding (RLE) is a lossless form of data compression.

Sequences having same value occurring many times are encoded to store only a single value and the number of times it occurs.

This greatly decreases the number of bits and file size, especially on large files, or when storage space is limited.
