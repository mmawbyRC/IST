Run Length Encoding

RLE, or Run Length Encoding, is a form of compression that turns colours and pixels into binary. The more colours there are, the more bits there will be as a result. Watch this video to find out more.

Bitmaps

Bitmaps are a type of visual element that are created through the use of pixels. These pixels are ordered to make a picture. Pixels is short for picture element. This picture below is an example of a bitmap with the pixels ordered to create the shape of the letter 'a'.

Text

Text is another web element that is used to convey information. Any writing or information on a page is classified as text. It can be used in textboxes, lists, tables, etc. There are also many different fonts. See the video below for more information on fonts.
