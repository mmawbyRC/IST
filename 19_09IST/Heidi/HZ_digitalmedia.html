Digital Media

Text

Text is able to be digitized by electronic signals that the computer receives once a key is pressed. The computer reads the signal and converts it into a symbol that is shown as a letter or number or symbol.

Bitmaps

A bitmap is an an image created from lots of bits in rows, each one a single colour. It is also called a raster graphic.

Run Length Encoding

Run length encoding is a simple form of data compression. It is lossless and runs on strings having the same value occurring many times in a row. It encodes the string to have a single value.
